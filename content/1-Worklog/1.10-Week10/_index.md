### Week 10 Objectives:

- Deploy and manage workloads using Red Hat OpenShift Service on AWS (ROSA) and set up an introductory CI/CD workflow.
- Construct and evaluate an analytics ecosystem on AWS using streaming ingestion, Glue, EMR, Athena, QuickSight, and Redshift.
- Develop business dashboards and monitor account and network activity through Amazon QuickSight, VPC Flow Logs, and delegated billing permissions.
- Build and deploy cloud applications using AWS CDK, SNS/SQS-driven event workflows, and Lambda-based serverless components.
- Create complete serverless web stacks with API Gateway, Lambda, DynamoDB, Cognito, and CloudFront, with SSL and authentication support.
- Implement order-processing pipelines and CI/CD processes for serverless solutions using SQS, SNS, SAM, and CodePipeline.

### Week 10 Achievements:

- Completed application deployment using ROSA by enabling the service, provisioning a cluster, deploying workloads, and creating a functional CI/CD setup with CodeCommit, CodeBuild, and CodePipeline.

- Built an end-to-end analytics platform:

  - Stream ingestion with Kinesis Firehose and metadata cataloging via Glue Crawlers.
  - Data transformation using Glue tools, DataBrew, and large-scale EMR processing.
  - Query and analysis in Athena, real-time analytics via Kinesis Data Analytics, and visualization in QuickSight.
  - Served final outputs using Lambda and organized data warehousing with Redshift.

- Improved visualization and observability:

  - Designed dynamic QuickSight dashboards with various chart types, formatting, and filters.
  - Enabled and examined VPC Flow Logs within CloudWatch Logs for network insight.
  - Delegated billing console access using IAM groups and custom policies.

- Developed infrastructure and event-driven systems using AWS CDK, SNS, and SQS:

  - Set up a CDK environment with EC2 and IAM, and built architectures integrating API Gateway, ELB, ECS, Lambda, and S3.
  - Implemented SNSâ€“SQS messaging patterns with advanced filtering.
  - Created Lambda-based image processing and DynamoDB workflows.

- Built full serverless stacks:

  - Deployed front-end apps integrated with API Gateway, Lambda, and DynamoDB.
  - Implemented authentication with Cognito and validated workflows using UI tools.
  - Secured deployments using Route 53 domains, ACM certificates, and CloudFront over HTTPS.

- Implemented order workflows and CI/CD automation:
  - Developed an SQS/SNS-powered order routing system backed by DynamoDB and Lambda.
  - Built CI/CD pipelines for SAM backends and front-end applications using CodePipeline and Git-based triggers.
